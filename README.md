# Michelle & Tom Wedding Invitation

Static bilingual wedding invitation page (English default + Traditional Chinese).

Site root is this folder (`index.html` at the top level) — ready for Vercel as a static site.

## Open locally

Open `index.html` in a browser, or serve the folder:

```bash
npx --yes serve .
```

## Deploy to Vercel

Any of these works; no build step required (Framework Preset: **Other** / static).

1. **GitHub + Vercel (recommended)**  
   - Create a GitHub repo and push this folder.  
   - In [vercel.com](https://vercel.com), **Add New Project** → import the repo.  
   - Leave Root Directory as `.` and deploy.

2. **Vercel CLI** (after installing Node.js)  
   ```bash
   npm i -g vercel
   vercel
   ```  
   Follow the prompts; use `vercel --prod` for production when ready.

3. **Drag and drop**  
   In the Vercel dashboard, use **Add New… → Project** and upload this folder.

`vercel.json` enables clean URLs and light caching headers for `/images`.
